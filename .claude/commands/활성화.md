# /활성화 — Issue 활성화 및 파이프라인 시작/재개

Linear Issue의 파이프라인을 시작하거나 재개한다.

## $ARGUMENTS

Linear ID (예: `PRJ-47`). 미지정 시 Linear MCP로 활성 Issue 목록 조회 후 사용자 선택.

---

## 실행

<!-- 참조 문서용 — 실행 경로가 아닌 스킬 참조 -->
dev-pipeline Skill을 호출한다.

| 단계 | 행위 |
|------|------|
| 1 | `$ARGUMENTS`에서 Linear ID 파싱. 미지정 시 Linear MCP로 활성 Issue 목록 조회 → `AskUserQuestion`으로 사용자 선택 |
| 2 | dev-pipeline Skill 호출 — Linear 상태 조회 → type별 워크플로우 분기 → 해당 스킬 실행 |
