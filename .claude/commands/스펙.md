# /스펙 — 기능 명세 생성

기능 단위의 spec 문서를 생성한다. 조사 → 요구사항 인터뷰 → spec 문서 출력.

## $ARGUMENTS

기능 제목 (선택). 예: `전투 시스템`. 미지정 시 `AskUserQuestion`으로 입력 요청.

---

## 실행

<!-- 참조 문서용 — 실행 경로가 아닌 스킬 참조 -->
spec Skill을 호출한다.

| 단계 | 행위 |
|------|------|
| 1 | `$ARGUMENTS`에서 기능 제목 파싱 |
| 2 | 기능 제목이 없으면 `AskUserQuestion`으로 입력 요청 |
| 3 | spec Skill 호출 — 조사 → 인터뷰 → spec 문서 생성 |

> spec Skill 상세: [.claude/skills/spec/SKILL.md](../skills/spec/SKILL.md) — 참조 문서용 (실행 경로가 아님)
