# /조사 — 사전 조사

입력 자료를 분석하고 조사 보고서를 생성한다.

## $ARGUMENTS

조사 주제 또는 Linear ID (선택). 미지정 시 사용자에게 주제 입력 요청.

---

## 실행

investigation Skill을 호출한다.

| 단계 | 행위 |
|------|------|
| 1 | `$ARGUMENTS`에서 Linear ID 또는 조사 주제 파싱 |
| 2 | Linear ID가 있으면 해당 Issue의 조사 수행 |
| 3 | Linear ID가 없으면 독립 조사 모드 — 결과를 `docs/shared/` 또는 지정 경로에 저장 |
| 4 | investigation Skill 호출 — 조사 수행 → 보고서 생성 |

> investigation Skill 상세: [.claude/skills/investigation/SKILL.md](../skills/investigation/SKILL.md)
