# /조사 — 사전 조사

Linear Issue에 대한 조사를 수행하고 보고서를 생성한다.

## $ARGUMENTS

Linear ID (필수). 예: `PRJ-47`. 미지정 시 `AskUserQuestion`으로 입력 요청.

> Linear ID 없이 조사가 필요하면 먼저 `/등록`으로 research type Issue를 생성하세요.

---

## 실행

<!-- 참조 문서용 — 실행 경로가 아닌 스킬 참조 -->
investigation Skill을 호출한다.

| 단계 | 행위 |
|------|------|
| 1 | `$ARGUMENTS`에서 Linear ID 파싱 |
| 2 | Linear ID가 없으면 `AskUserQuestion`으로 입력 요청. 입력 없으면 `/등록` 안내 후 중단 |
| 3 | investigation Skill 호출 — 조사 수행 → 보고서 생성 → feature-close 연계 |

> investigation Skill 상세: [.claude/skills/investigation/SKILL.md](../skills/investigation/SKILL.md)
