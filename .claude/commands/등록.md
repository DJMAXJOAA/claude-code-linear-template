# /등록 — Issue 등록

새 Issue를 Linear에 등록하고 Git 문서 폴더를 생성한다.

## $ARGUMENTS

type 힌트 (선택). 예: `feature 사용자 인증 기능`, `bug 로그인 실패`. 미지정 시 AI가 추천.

---

## 실행

gen-hub Skill을 호출한다.

| 단계 | 행위 |
|------|------|
| 1 | `$ARGUMENTS`에서 type + 제목/설명 파싱 |
| 2 | gen-hub Skill 호출 — Linear Issue 생성 + `docs/{type}/{LINEAR-ID}/_index.md` 생성 |
| 3 | 생성된 Issue 정보 출력 (LINEAR-ID, URL, type) |

> gen-hub Skill 상세: [.claude/skills/gen-hub/SKILL.md](../skills/gen-hub/SKILL.md)
