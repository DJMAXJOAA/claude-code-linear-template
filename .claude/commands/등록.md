# /등록 — Issue 등록

새 Issue를 Linear에 등록하고 Git 문서 폴더를 생성한다.

## $ARGUMENTS

자유 형식 텍스트. 입력 패턴에 따라 gen-hub Skill이 단건/배치 모드를 자동 판별한다.

- **단건**: type 힌트 + 짧은 제목. 예: `feature 사용자 인증 기능`
- **배치**: 긴 텍스트, 복수 기능 서술
- **--from-spec**: `--from-spec {spec-name}` — spec FR 기반 배치 생성
- **미지정**: `AskUserQuestion`으로 입력 요청

---

## 실행

| 단계 | 행위 |
|------|------|
| 1 | `$ARGUMENTS` 파싱 — 단건/배치/--from-spec 모드 판별 |
| 2 | gen-hub Skill 호출 (모드 + 입력 텍스트 전달) |
| 3 | 생성 결과 출력 (LINEAR-ID, URL, type) |

> gen-hub Skill 상세: [.claude/skills/gen-hub/SKILL.md](../skills/gen-hub/SKILL.md)
