# /등록 — Issue 등록

새 Issue를 Linear에 **단건** 등록하고 Git 문서 폴더를 생성한다. 일괄 등록은 지원하지 않는다.

## $ARGUMENTS

type 힌트 (선택). 예: `feature 사용자 인증 기능`, `bug 로그인 실패`. 미지정 시 `AskUserQuestion`으로 type + 제목 입력 요청.

---

## 실행

<!-- 참조 문서용 — 실행 경로가 아닌 스킬 참조 -->
gen-hub Skill을 호출한다.

| 단계 | 행위 |
|------|------|
| 1 | `$ARGUMENTS` 미지정 시 `AskUserQuestion`으로 type + 제목/설명 입력 요청 |
| 2 | `$ARGUMENTS`에서 type + 제목/설명 파싱 |
| 3 | gen-hub Skill 호출 — Linear Issue 생성 + `docs/{type}/{LINEAR-ID}/_index.md` 생성 |
| 4 | 생성된 Issue 정보 출력 (LINEAR-ID, URL, type) |

> gen-hub Skill 상세: [.claude/skills/gen-hub/SKILL.md](../skills/gen-hub/SKILL.md)
