# /현황 — Linear 활성 Issue 조회

> **조회 커맨드** — 4단계 게이트 면제. 읽기 전용.

Linear MCP로 현재 활성 Issue 목록을 조회하고 포맷팅하여 출력한다.

## $ARGUMENTS

필터 (선택). `feature` / `bug` / `improvement` / `all`. 미지정 시 `all`.

---

## 실행

| 단계 | 행위 |
|------|------|
| 1 | Linear MCP로 팀의 활성 Issue 목록 조회 (Backlog ~ In Review 상태, 최대 20건) |
| 2 | `$ARGUMENTS` 필터 적용 (type Label 기반) |
| 3 | 상태별 그룹핑 + Priority 순 정렬하여 테이블 출력 |

## 출력 형식

```
## 활성 Issue 현황 (최대 20건)

### In Progress (2)
| ID | 제목 | Type | Priority | Assignee |
|----|------|------|----------|----------|
| MYP-1 | (제목) | feature | High | @user |
| MYP-3 | (제목) | improvement | Medium | @user |

### Planning (1)
| ID | 제목 | Type | Priority | Assignee |
|----|------|------|----------|----------|
| MYP-5 | (제목) | feature | High | — |

### Backlog (3)
...
```
