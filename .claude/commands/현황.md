# /현황 — Linear 활성 Issue 조회

Linear MCP로 현재 활성 Issue 목록을 조회하고 포맷팅하여 출력한다.

## $ARGUMENTS

필터 (선택). `feature` / `bug` / `improvement` / `research` / `all`. 미지정 시 `all`.

---

## 실행

| 단계 | 행위 |
|------|------|
| 1 | Linear MCP로 팀의 활성 Issue 목록 조회 (Backlog ~ Verifying 상태) |
| 2 | `$ARGUMENTS` 필터 적용 (type Label 기반) |
| 3 | 상태별 그룹핑하여 테이블 출력 |

## 출력 형식

```
## 활성 Issue 현황

### In Progress (2)
| ID | 제목 | Type | Assignee |
|----|------|------|----------|
| PRJ-47 | 이모티콘 핑 시스템 | feature | @user |
| PRJ-50 | API 응답 캐싱 | improvement | @user |

### Planning (1)
| ID | 제목 | Type | Assignee |
|----|------|------|----------|
| PRJ-52 | 사용자 인증 | feature | — |

### Backlog (3)
...
```
