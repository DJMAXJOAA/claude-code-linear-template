## 프레임워크 검증

프레임워크 파일의 구조적 무결성을 검증한다.

| 카테고리 | 검증 내용 |
|----------|---------|
| **구조** | CLAUDE.md, Rules, Skills, Commands, Guides 파일 존재 확인 |
| **참조** | CLAUDE.md 인덱스 경로 vs 실제 파일 일치 |
| **규칙** | CLAUDE.md 줄 수 제한 (200줄), paths frontmatter 정합성 |
| **일관성** | 스킬 간 MCP 도구명 일관성, 커맨드→스킬 매핑 정합성 |

| 단계 | 행위 |
|------|------|
| 1 | CLAUDE.md 읽기 — Key File Index에서 참조 경로 목록 추출 |
| 2 | 각 참조 경로의 실제 파일 존재 확인 |
| 3 | Rules 파일의 paths frontmatter vs CLAUDE.md Rules Delegation 테이블 비교 |
| 4 | CLAUDE.md 줄 수 확인 |
| 5 | 검증 결과 테이블 출력 (PASS/FAIL + 근거) |
