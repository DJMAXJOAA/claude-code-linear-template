# /검증 — 프레임워크 구조 무결성 검증

프레임워크 파일의 구조적 무결성을 검증한다. 파일 존재성, 참조 일관성, 규칙 준수를 확인.

## $ARGUMENTS

검증 범위 (선택). `전체` / `참조` / `구조`. 미지정 시 전체 검증.

---

## 검증 항목

| 카테고리 | 검증 내용 |
|----------|---------|
| **구조** | CLAUDE.md, Rules, Skills, Commands, Guides 파일 존재 확인 |
| **참조** | CLAUDE.md 인덱스 경로 vs 실제 파일 일치 |
